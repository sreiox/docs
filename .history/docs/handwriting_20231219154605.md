## 浅拷贝和深拷贝是什么？手写一个深拷贝

浅拷贝是会将对象的每个属性进行依次复制，但是当对象的属性值是`引用类型`时，实质复制的是其`引用`，当引用指向的值改变时也会跟着变化。 深拷贝复制`变量值`，对于引用数据，则递归至基本类型后，再复制。 深拷贝后的对象与原来的对象是`完全隔离`的，`互不影响`，对一个对象的修改并不会影响另一个对象

![tup]

```js
function deepClone(obj){
    if(typeof obj !== 'object' || typeof obj === null) return obj;

    const result = obj instanceof Array? []: {}

    for(let key in obj){

        result[key] = deepClone(obj[key])
    } 
    return result;
}

const obj = {a: 1, b: {c:3}, d: 5}
const objClone = deepClone(obj)
obj.a = 123
console.log(obj) //{a: 123, b: {…}, d: 5}
console.log(objClone) //{a: 1, b: {…}, d: 5}
```
